# SaaS Ticket Management System

## 🚀 Overview
A complete SaaS platform for ticket management with organizations, projects, and team collaboration.

## 📋 Features
- 🔐 Authentication & Authorization (JWT)
- 🏢 Multi-tenant Organizations
- 👥 Role-Based Access Control
- 📊 Project Management
- 🎫 Ticket Management (Issue Tracking)
- 💬 Comments & Collaboration
- 📎 File Attachments
- 🔔 Notifications
- 📈 Dashboard Analytics
- 🏃 Sprint Management
- 🔍 Global Search

## 🛠️ Tech Stack

### Backend
- Django 4.x
- Django REST Framework
- PostgreSQL
- Redis (for caching)
- JWT Authentication

### Frontend
- React.js
- Axios
- Tailwind CSS

### DevOps
- Docker & Docker Compose
- Git
- GitHub/GitLab
